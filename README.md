# Portfolio

## Deploying

If need-be, alter `script/deploy.sh` to reflect your correct username, server, and domain.

If you don't have an ssh key already generated, run

```
ssh-keygen -t rsa -b 2048
```

Then,

```
ssh-copy-id username@server
```

Finally,

```
./script/deploy.sh
```
